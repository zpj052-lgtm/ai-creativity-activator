# Multi-Source Collision Machine (全领域全源碰撞机器) 🚀

> **让“自然规律”、“人文学科”、“社会热点”、“神话传说”与“万物”碰撞，激发跨时空的创新灵感。**

[English](./README_EN.md) | [中文](./README.md)

## 🌟 核心理念

大多数人的思维局限在自己的专业领域（专业隔离）。
本项目是一个基于 DeepSeek 开发的**多智能体创新引擎**。它不直接思考问题，而是先从物理学、人文学科、甚至神话笑话中挖掘底层概念，强制将其与您的目标难题进行逻辑对齐。

这种“强制跨界”能有效破除“领域依赖性”，生成具有深层逻辑支撑的原创点子。

### 系统架构
1.  **概念搜索者 (Concept Finder)**: 从科学、人文、新闻、神话、笑话、神评等 6 大维度挖掘素材。
2.  **内容碰撞者 (Problem Collider)**: 将非相关的素材强行映射到您的目标难题，生成全新的策略或情节。
3.  **逻辑裁判 (Idea Judge)**: 从逻辑、创新和实战潜力三个维度对产出进行严苛打分。

## 🛠️ 安装与使用

### 1. 环境准备
```bash
git clone https://github.com/your-username/multi-source-collision.git
cd multi-source-collision
pip install -r requirements.txt
```

### 2. 配置 API Key
在系统中设置环境变量 `DEEPSEEK_API_KEY`。

### 3. 开始碰撞
- **混合碰撞模式** (推荐):
  ```bash
  python collision_machine_mixed.py --iterations 10 --problem "您的难题"
  ```
- **文学/情节模式**:
  ```bash
  python collision_machine_novel.py --iterations 5
  ```
